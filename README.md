# Визуализатор землетрясений

Интерактивное веб-приложение для анализа сейсмических событий и сетей станций в Центральной Азии. Построено на Streamlit и Folium.

## Возможности

- Интерактивная карта с маркерами землетрясений, размер которых определяется магнитудой (Ml) или классом энергии (K)
- Цветовая кодировка по глубине с легендой
- Фильтрация по диапазону дат с валидацией
- Пространственная фильтрация — прямоугольник или круг
- Включение/отключение сетей сейсмических станций
- Отображение реперной точки НС РАН
- Несколько стилей карты: светлая, тёмная, улицы, спутник, топо
- Таблица данных с сортировкой
- Вкладка со статистикой и графиками
- Вкладка со списком станций

## Требования

- Python 3.9+
- Зависимости указаны в `requirements.txt`

## Установка

**1. Клонировать репозиторий**

```bash
git clone https://github.com/nishonow/internship
cd internship
```

**2. Создать виртуальное окружение**

```bash
python -m venv .venv
```

Активировать:

- Windows: `.venv\Scripts\activate`
- macOS / Linux: `source .venv/bin/activate`

**3. Установить зависимости**

```bash
pip install -r requirements.txt
```

## Запуск приложения

```bash
streamlit run app.py
```

Приложение автоматически откроется по адресу `http://localhost:8501`.

## Формат данных

### Файл землетрясений (обязательный)

Загрузите файл `.xlsx` со следующими колонками:

| Колонка | Тип | Описание |
|---------|-----|----------|
| `Origin` | datetime | Дата и время события (`YYYY-MM-DD HH:MM:SS`) |
| `Lat` | float | Широта |
| `Lon` | float | Долгота |
| `Depth` | float | Глубина в км |
| `Ml` | float | Местная магнитуда |
| `K` | float | Класс энергии (необязательный) |

### Файл станций (необязательный)

Загрузите отдельный файл `.xlsx` со следующими колонками:

| Колонка | Тип | Описание |
|---------|-----|----------|
| `Network` | string | Код сети (например, `KR`, `GE`, `KN`) |
| `Station_code` | string | Код станции |
| `Lat` | float | Широта |
| `Lon` | float | Долгота |
| `Elevation` | float | Высота в метрах (необязательный) |
| `Info` | string | Дополнительная информация о станции (необязательный) |
